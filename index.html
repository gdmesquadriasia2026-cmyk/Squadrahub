🏦 Integração Bancária — Contas e APIs
🔗 Como integrar com seu banco
1. Acesse o internet banking PJ do seu banco e ative a API de cobranças
2. Gere as credenciais: Client ID, Client Secret e/ou Token de acesso
3. Configure o Webhook URL no painel do banco para receber baixas automáticas
4. Cole as credenciais abaixo e clique em Testar Conexão
Selecione os bancos que deseja integrar:
🔒 As credenciais ficam salvas apenas neste dispositivo (localStorage). Para produção, recomenda-se um backend seguro.
📡 Webhook URL (para configurar no banco): https://seu-dominio.com/api/webhook/banco
📋 Bancos compatíveis: Banco Inter, Asaas, PagSeguro, Mercado Pago, Sicredi, BB, Itaú, Bradesco, Santander, Caixa, Sicoob, PagBank, Gerencianet/Efí

⚠️ Integração com SEFAZ e Prefeitura
Para emitir NFs válidas, é necessário integrar com:
